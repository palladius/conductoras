# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-29
### Added
- Dynamic Repo Chooser UI powered by a lightweight Python backend (`bin/arcade_server.py`).
- Automatic Gravatar fetching using MD5 hashed author emails.
- Parametric static deployment builder (`bin/build_static.py`) for GitHub Pages support.
- Visuals: Branch emojis (`🌿`), scrolling horizontal Tag Lines for major milestones, and excluded git stashes.

## [0.0.1] - 2026-06-28
### Added
- Initial release.
